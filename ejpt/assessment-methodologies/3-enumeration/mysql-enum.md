# MYSQL Enum

**`MYSQL`** - 

A client machine makes a `request` to a server (usually from a website), which then sends a `response` message back to the client.

Default `HTTP` port is **`80`** and `HTTPS` port is **`443`**.

```bash
sudo nmap -p80 -sV -O <TARGET_IP>
```
